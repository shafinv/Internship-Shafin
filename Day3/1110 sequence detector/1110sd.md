# 1110 Sequence Detector Using Verilog HDL

A Verilog HDL implementation of a **Moore Finite State Machine (FSM)** that detects the binary sequence **1110** in a serial input stream. When the sequence is detected, the output signal `detected` is asserted.

---

## 📖 Overview

A Sequence Detector is a sequential circuit used to identify a specific pattern of bits from a serial input stream.

This project implements a detector for the sequence:

```text
1110
```

The design is based on a Finite State Machine (FSM) and continuously monitors the input stream. Whenever the sequence **1110** is detected, the output `detected` becomes HIGH.

---

## ✨ Features

* Detects the sequence **1110**
* FSM-based design
* Positive-edge-triggered operation

* Synthesizable Verilog HDL code

## 🧠 Theory

A Sequence Detector monitors incoming bits and transitions through different states based on the received input.

For detecting **1110**, the FSM progresses through the following stages:

```text
1
11
111
1110
```

Once the complete sequence is received, the detector output becomes HIGH.

---

## 📋 State Description

| State | Meaning                 |
| ----- | ----------------------- |
| Idle  | Initial state           |
| S1    | First '1' received      |
| S2    | Sequence '11' received  |
| S3    | Sequence '111' received |

When the FSM is in **S3** and receives a **0**, the sequence **1110** is detected.

## 📄 Verilog Implementation

### State Encoding

```verilog
parameter idle = 2'b00;
parameter s1   = 2'b01;
parameter s2   = 2'b10;
parameter s3   = 2'b11;
```

### Present State Logic

```verilog
always @(posedge clk)
begin
    if(rst)
        ps <= idle;
    else
        ps <= ns;
end
```

### Detection Logic

```verilog
s3:
begin
    if(din == 0)
    begin
        ns = idle;
        detected = 1'b1;
    end
end
```
